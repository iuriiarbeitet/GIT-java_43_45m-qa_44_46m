1.Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory)?
	git checkout -- <file>

2.Как вернуть файл из индекса в рабочий каталог?
	git reset HEAD <file>

3.Как отменить ошибочный commit ?
	git revert <commit-hash>

4.Как создать новую ветку и переключится в нее?
	git branch <branch-name>
	git checkout <branch-name>

5.В чем отличение между fast-forward merge  and non fast-forfard merge?
fast-forward merge - это слияние, когда одна ветка является прямым 
предшественником другой и Git просто перемещает указатель. 
non fast-forward merge - это слияние, когда две ветки имеют общую 
историю и Git создает новый коммит с двумя родителями.

6.Когда может возникнуть конфликт слияния веток, как  его разрешить?
Конфликт слияния веток возникает, когда одна и та же часть кода 
или файла изменена в разных ветках. Чтобы решить конфликт, нужно 
вручную редактировать файлы, выбрать изменения и сделать коммит.

7.Можно ли удалить ветку? Как? 
	Да, ветку можно удалить. Команда: git branch -d <branch-name>.
